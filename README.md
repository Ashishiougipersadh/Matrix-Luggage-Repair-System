# Matrix - Luggage Repair System
A cross-platform luggage repair management system built with Flutter, designed to streamline communication and repair workflows between customers and staff.

## Project Overview
The Luggage Repair System is a Flutter-based application that allows customers to request, book, and track luggage repairs while enabling staff members to efficiently manage repair jobs. The system focuses on improving transparency, communication, and convenience throughout the luggage repair process.

Customers can submit repair requests, schedule repairs, and arrange pickup or delivery of their luggage. Staff members are able to manage incoming repair requests, update repair statuses, and communicate directly with customers through a real-time messaging feature.

##  Key Features

### Customer Features
- Submit luggage repair requests
- Book repair appointments
- Arrange luggage pickup and delivery
- Track repair progress
- Real-time chat with a staff member for updates and inquiries

### Staff Features
- View and manage customer repair requests
- Update repair status and booking details
- Communicate with customers via real-time messaging
- Organize and oversee luggage repair workflows
